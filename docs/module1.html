Module 1: Getting Setup in the R and Rstudio Environment

Both R and RStudio are cross platform compatible (Windows/Mac/Linux) and FREE. Make sure to install BOTH R and RStudio to practice R and do the exercises in this mini course. Choose the appropriate download for your OS before downloading. If you already have these installed, check to make sure you are up-to-date with the latest versions of both!
